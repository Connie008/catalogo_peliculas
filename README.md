# Catálogo de Películas

Aplicación móvil desarrollada con Flutter y Firebase.

## Funciones
- Registro de usuarios
- Inicio de sesión
- Catálogo de películas
- Alta y baja de películas
- Visualización de detalles

## Tecnologías
- Flutter
- Firebase Authentication
- Cloud Firestore
- Dart
